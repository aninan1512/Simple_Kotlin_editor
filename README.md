
# Kotlin Code Editor

A lightweight desktop application designed for editing Kotlin code. This JavaFX-based editor includes real-time syntax highlighting, file operations, and basic editing tools to provide a distraction-free coding environment for beginners.

---

## **Features**
- **Real-Time Syntax Highlighting:** Color-coded keywords, variables, and symbols for better readability.
- **File Operations:** Open and save `.kt` files easily.
- **Editing Tools:** Undo, redo, copy, cut, and paste for smooth text manipulation.
- **Clean User Interface:** A simple and intuitive design for beginners.

---

## **Requirements**
To run this program, you need the following:
1. **Java Development Kit (JDK):** Version 21 or compatible with JavaFX SDK.
2. **JavaFX SDK:** Download from [OpenJFX](https://openjfx.io/).
3. **Build Tool :** Gradle or Maven for dependency management.
4. **IDE :** IntelliJ IDEA, Eclipse, or any Java-supported editor.

---

## **Setup Instructions**

### **Step 1: Download JavaFX SDK**
1. Visit [OpenJFX Downloads](https://openjfx.io/) and download the JavaFX SDK for your operating system.
2. Extract the contents to a directory (e.g., `C:/javafx-sdk-21`).
